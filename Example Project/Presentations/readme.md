This folder will contain all presentations that are created for the project, both internal and external. The folder structure that should be used is included in this example.

The AU Internal Presentations folder is for events like URFA, CSM Departmental Presentations, AU Outreach, or Senior Presentations.

The External Conference Presentations folder is for all presentations given at events external to AU, such as ACS or GAWP.

The Internal Updates folder is for all presentations created for updates internal to the research group.
