This example project is intended to serve as the model for each projects individual folder. Each project must contain all of the folders listed along with a readme.md file for the project. 

The readme file should contain, at a minimum, the following:
- A full description of the project that clearly identfies the prupose of project and the need it is fulfilling.
- The name(s) of the researchers currently working on the project and their University emails
- The name and University email of the project supervisor.

You are free to add other things to the readme file, have fun with it and make it your own!
